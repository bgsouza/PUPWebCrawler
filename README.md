# PUPWebCrawler

Engine para fazer captura de dados em sistes simple ou com navegação dinâmica (ajax)

## Dependências
  - NodeJS >= 8
  - PM2
  
## Para instalar o projeto:
-  na raiz executar: npm install
-  executar: npm i -g pm2

## Executar o projeto:
- npm start
